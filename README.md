# Amazon-scrap-dashboad
1. Select a category and URL structure. 
2. Send request using requests.get() 
3. Parse HTML using BeautifulSoup. 
4. Extract desired tags/classes 
5. Store scraped data into DataFrame. 
6. Clean values. 
7. Perform EDA. 
8. Create dashboards. 
Data Cleaning and Preprocessing 
Remove missing/duplicate values. 
Convert price strings to float. 
Normalize ratings/review counts. 
Remove HTML escape characters or tags. 
Export cleaned data to CSV.
